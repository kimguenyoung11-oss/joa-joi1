Come pubblicare GRATIS (GitHub Pages)
------------------------------------
1) Crea un account GitHub e un repository nuovo chiamato: joa-joi-site
2) Carica TUTTI i file della cartella in root del repository.
3) Vai su Settings → Pages → Source = "Deploy from a branch", Branch = "main", /root → Save
4) Attendi 1–2 minuti e apri l'URL che GitHub indica. Il sito è online.

Ordini via WhatsApp (versione GRATIS)
------------------------------------
- In script.js cerca 'wa.me/390000000000' e sostituisci con il numero corretto (es. +39XXXXXXXXXX senza +).
- Ogni card e pagina prodotto hanno il pulsante "Ordina su WhatsApp".

Passare a WordPress + WooCommerce (pagamenti online)
----------------------------------------------------
1) Acquista dominio + hosting (es: piano base Linux + SSL).
2) Installa WordPress dal pannello hosting (one-click).
3) Installa WooCommerce e configura:
   - Valuta EUR, Italia come paese, tasse, spedizioni.
   - Pagamenti: Stripe (carta), PayPal, bonifico.
4) Scegli un tema (Astra/Blocksy) + importer demo shop minimal.
5) Crea prodotti con: nome, prezzo, descrizione, immagini (usa le .jpg generate qui o foto reali).
6) Imposta pagine: Home, Shop, Carrello, Checkout, Account.

SEO e dettagli rapidi
---------------------
- Title e meta description per ogni pagina.
- Immagini: peso < 300 KB, dimensioni 1200x1600 per prodotto.
- Favicon e logo già presenti (assets/).
